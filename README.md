# ISODash

**Automated ISO Performance Indicators Calculation System**

ISODash is a Flutter Web application that automates the calculation of 3 critical ISO performance indicators by extracting data from OpenProject via API, replacing manual Excel workflows with automated KPI calculation, cause tracking, and PDF/DOCX export capabilities.
